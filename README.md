# Jewelry Shop

Веб-приложение на Spring Boot для покупки и продажи ювелирных украшений.
Поддерживает две роли: **мастер** (создаёт и управляет товарами) и **покупатель** (просматривает каталог, добавляет в корзину, оформляет заказы).

## Стек технологий

- **Backend:** Spring Boot, Spring Security, Spring Data JPA
- **База данных:** PostgreSQL
- **Кеш:** Redis
- **Шаблонизатор:** FreeMarker
- **Документация API:** Swagger UI
- **Развёртывание:** Docker, Docker Compose

## Функционал

- Регистрация и авторизация с разграничением ролей (мастер / покупатель)
- Каталог товаров с фильтрацией по категории и диапазону цен
- Конвертация цен в USD и EUR через внешний API 
- Корзина и оформление заказов без перезагрузки страницы 
- Отзывы на товары
- Кеширование данных через Redis 
- REST API с документацией через Swagger UI 
- Защита от XSS, CSRF
